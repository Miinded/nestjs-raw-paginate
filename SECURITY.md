# Security Policy

## Supported versions

The latest published version of `@miinded/nestjs-raw-paginate` receives security fixes.

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities.

Send a report with:

- package version
- reproducible steps
- impact assessment
- proposed remediation if available

Contact: dev@miinded.com

We will acknowledge receipt within 72 hours and provide follow-up status.
